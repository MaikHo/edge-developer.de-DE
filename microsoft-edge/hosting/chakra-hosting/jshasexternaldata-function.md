# JsHasExternalData-Funktion
Bestimmt, ob ein Objekt ein externes Objekt ist.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsHasExternalData(  
   _In_ JsValueRef object,  
   _Out_ bool *value  
);  
```  
  
#### Parameter  
 `object`  
 Das Objekt.  
  
 `value`  
 Gibt an, ob es sich um ein externes Objekt handelt.
