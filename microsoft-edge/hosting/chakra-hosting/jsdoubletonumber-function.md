# JsDoubleToNumber-Funktion
Erstellt einen Zahlenwert aus einem `double` Wert.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsDoubleToNumber(  
   _In_ double doubleValue,  
   _Out_ JsValueRef *value  
);  
```  
  
#### Parameter  
 `doubleValue`  
 Die `double` Konvertierung in einen Zahlenwert.  
  
 `value`  
 Der neue Zahlenwert.
