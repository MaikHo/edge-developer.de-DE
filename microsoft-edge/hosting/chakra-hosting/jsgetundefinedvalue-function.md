# JsGetUndefinedValue-Funktion
Ruft den Wert `undefined` im aktuellen Skriptkontext ab.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsGetUndefinedValue(  
   _Out_ JsValueRef *undefinedValue  
);  
```  
  
#### Parameter  
 `undefinedValue`  
 Der `undefined` Wert.

 Erfordert einen aktiven Skriptkontext.
