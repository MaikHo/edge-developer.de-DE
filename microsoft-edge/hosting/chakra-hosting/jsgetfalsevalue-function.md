# JsGetFalseValue-Funktion
Ruft den Wert `false` im aktuellen Skriptkontext ab.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsGetFalseValue(  
   _Out_ JsValueRef *falseValue  
);  
```  
  
#### Parameter  
 `falseValue`  
 Der `false` Wert.
