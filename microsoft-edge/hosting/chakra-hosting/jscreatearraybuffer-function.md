# JsCreateArrayBuffer-Funktion
Erstellt ein JavaScript `ArrayBuffer` -Objekt.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateArrayBuffer(  
   _In_ unsigned int byteLength,  
   _Out_ JsValueRef *result  
);  
```  
  
#### Parameter  
 `byteLength`  
 Die Anzahl der Bytes in der ArrayBuffer.  
  
 `result`  
 Das neue ArrayBuffer-Objekt.  
  
## Rückgabewert  
 Der Code `JsNoError` , wenn der Vorgang erfolgreich war, andernfalls ein Fehlercode.  
  
## Hinweise  
 Erfordert einen aktiven Skriptkontext.  
  
 Diese API wird nur im Microsoft Edge-Modus unterstützt.  
  
## Anforderungen  
 **Kopfzeile:** jsrt. h
