# JsGetGlobalObject-Funktion
Ruft das globale Objekt im aktuellen Skriptkontext ab.  
  
## Syntax  
  
```cpp  
STDAPI_(JsErrorCode) JsGetGlobalObject(  
   _Out_ JsValueRef *globalObject  
);  
```  
  
#### Parameter  
 `globalObject`  
 Das globale Objekt.
