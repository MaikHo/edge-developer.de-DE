## <span data-ttu-id="9aa50-121">Grundlegendes zum Browser, bevor Erweiterungen eingeführt werden</span><span class="sxs-lookup"><span data-stu-id="9aa50-121">Understanding the browser before Extensions are introduced</span></span>  

### <span data-ttu-id="9aa50-122">Jede Registerkarte des Browsers ist von jeder anderen Registerkarte isoliert</span><span class="sxs-lookup"><span data-stu-id="9aa50-122">Each browser tab is isolated from every other tab</span></span>  

<span data-ttu-id="9aa50-123">Um zu verstehen, was eine Microsoft Edge \ (Chromium \)-Erweiterung ist, müssen wir zunächst vollständig verstehen, was ein Multi Tab-Browser, wie Microsoft Edge, in erster Linie tut.</span><span class="sxs-lookup"><span data-stu-id="9aa50-123">To understand what a Microsoft Edge \(Chromium\) Extension is, we first need to fully understand what a multi tab browser, like Microsoft Edge does primarily.</span></span>  <span data-ttu-id="9aa50-124">Um zu beginnen, wird jede Browserregister Karte in einem einzelnen Thread ausgeführt, der Sie effektiv von anderen Browser Registern \ (oder Threads \) isoliert.</span><span class="sxs-lookup"><span data-stu-id="9aa50-124">To start, each browser tab runs in an individual thread that effectively isolates it from other browser tabs \(or threads\).</span></span>

### <span data-ttu-id="9aa50-126">Jede Registerkarte verarbeitet eine GET-Anforderung.</span><span class="sxs-lookup"><span data-stu-id="9aa50-126">Each tab handles one GET request</span></span>  

<span data-ttu-id="9aa50-127">Auf jeder Registerkarte wird im Wesentlichen die URL \ (auch als Uniform Resource Locator bezeichnet) verwendet, um einen einzelnen Datenstrom zu erhalten, der in der Regel ein HTML-Dokument ist.</span><span class="sxs-lookup"><span data-stu-id="9aa50-127">Each tab essentially uses the URL \(also known as the uniform resource locator\) to get a single stream of data which is typically an HTML document.</span></span>  <span data-ttu-id="9aa50-128">Dieser einzelne Datenstrom \ (oder die Seite \) enthält häufig Anweisungen (wie beispielsweise JavaScript-Tags, Bild Bezüge, CSS-Bezüge usw.).</span><span class="sxs-lookup"><span data-stu-id="9aa50-128">That single stream \(or page\), often includes instructions \(like JavaScript include tags, image references, CSS references, and more\).</span></span>  <span data-ttu-id="9aa50-129">Letztendlich werden alle benötigten Ressourcen auf eine Registerkartenseite heruntergeladen, und in der Regel wird eine Visualisierung angezeigt, die auf der Registerkarte Browser vollständig gerendert wird.</span><span class="sxs-lookup"><span data-stu-id="9aa50-129">Ultimately, all the resources needed are downloaded to that one tab page and typically a visualization appears which we see in the browser tab completely rendered.</span></span>  

### <span data-ttu-id="9aa50-130">Die gesamte Kommunikation von jeder Registerkarte erfolgt an Remoteserver.</span><span class="sxs-lookup"><span data-stu-id="9aa50-130">All communication from each tab is to remote servers</span></span>  

<span data-ttu-id="9aa50-131">Wenn Sie wissen, dass jede Registerkarte in einer isolierten Umgebung ausgeführt wird, bedeutet dies, dass diese Registerkarten voneinander isoliert sind, nicht aber das größere Internet.</span><span class="sxs-lookup"><span data-stu-id="9aa50-131">Understanding that each tab runs in an isolated environment means that these tabs are isolated from each other, but not the greater internet.</span></span>  <span data-ttu-id="9aa50-132">In der Regel werden diese Registerkarten, auf denen JavaScript als definierte Programmiersprache ausgeführt wird, an den Server zurückgegeben, der als ursprünglicher Server für die erste Get-Anforderung gedacht ist, die in die URL-Leiste oben auf der Registerkarte Browser eingegeben wurde.</span><span class="sxs-lookup"><span data-stu-id="9aa50-132">Typically, these tabs, running JavaScript as the defined programming language, communicate back to the server, that should be thought of as the originating server for that first GET request that was entered into the URL bar at the top of the browser tab.</span></span>  

## <span data-ttu-id="9aa50-133">Das Erweiterungsmodell dreht alles auf den Kopf</span><span class="sxs-lookup"><span data-stu-id="9aa50-133">The Extension model turns everything upside down</span></span>  

<span data-ttu-id="9aa50-134">Eine Erweiterung, genau wie Registerkarten, wird in einem einzelnen Thread ausgeführt, der vollständig von allen besprochenen Registerkarten Threads isoliert ist.</span><span class="sxs-lookup"><span data-stu-id="9aa50-134">An Extension, just like tab pages, runs in a individual thread which is completely isolated from all tab page threads that are discussed.</span></span>  <span data-ttu-id="9aa50-135">Im Gegensatz zu den Registerkarten, deren Aufgabe darin besteht, in der Regel eine einzelne Get-Anforderung an einen Remoteserver auszugeben, wird eine Visualisierung dieser Daten im Browser angezeigt, wobei es sich um den Server handelt, der sich zuvor auf dem anderen Ende der Internetverbindung befand, die auf einer Browserregister Karte ausgeführt wurde.</span><span class="sxs-lookup"><span data-stu-id="9aa50-135">Unlike the tabs whose job is to typically issue a single GET request to a remote server, then display a visualization of that data in the browser, the Extension, on the other hand is the server, that previously resided on the other end of the internet connection made from a browser tab.</span></span>  

![Erweiterungsmodell wandelt das Servermodell auf den Kopf](media/index-image3-upsidedown.png)  

<span data-ttu-id="9aa50-137">Das ist wirklich wichtig, um es zu verstehen.</span><span class="sxs-lookup"><span data-stu-id="9aa50-137">This is really important to understand.</span></span>  <span data-ttu-id="9aa50-138">Nachdem Sie eine Erweiterung erstellt und in Ihrem Browser installiert haben, haben Sie einen eigenständigen Webserver erstellt, der in Ihrem Browser lebt und atmet, aber immer noch von jeder Registerkarte isoliert ist, die in diesem Browser ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="9aa50-138">Once you create an Extension, and install it in your browser, you've created a standalone web server that is living and breathing inside of your browser but still isolated from every tab page running on that browser.</span></span>  

### <span data-ttu-id="9aa50-139">Das Extension Web Server-Bundle</span><span class="sxs-lookup"><span data-stu-id="9aa50-139">The Extension web server bundle</span></span>  

<span data-ttu-id="9aa50-140">Was ist eine Erweiterung?</span><span class="sxs-lookup"><span data-stu-id="9aa50-140">So what is an Extension?</span></span> <span data-ttu-id="9aa50-141">Hierbei handelt es sich um ein Bündel \ (oder als ZIP-Datei bezeichnet) von Webressourcen, die sich nicht von dem unterscheiden, was ein Web Developer auf einem Webserver veröffentlicht.</span><span class="sxs-lookup"><span data-stu-id="9aa50-141">It is a bundle \(or referred to as a zip file\) of web resources that are no different than what a web developer publishes to a web server.</span></span>  

<span data-ttu-id="9aa50-142">Diese ZIP-Datei enthält HTML, CSS, JavaScript, Bilder und alle erforderlichen Ressourcen, um eine Webseite zu erstellen.</span><span class="sxs-lookup"><span data-stu-id="9aa50-142">That zip file includes HTML, CSS, JavaScript, images and all the necessary assets to make a web page.</span></span>  <span data-ttu-id="9aa50-143">Es gibt jedoch eine zusätzliche Datei, die im Stammverzeichnis dieser zip-Datei erforderlich ist, und diese Datei hat den Namen `manifest.json` .</span><span class="sxs-lookup"><span data-stu-id="9aa50-143">There is however, one extra file that is required in the root of this zip file, and that file is named `manifest.json`.</span></span>  <span data-ttu-id="9aa50-144">Es handelt sich um die Blaupause für Ihre Erweiterung, die Dinge wie die Version ihrer Erweiterung, was ist der Titel, welche Privilegien benötigt Sie für die Ausführung und vieles mehr umfasst.</span><span class="sxs-lookup"><span data-stu-id="9aa50-144">It is the blueprint for your Extension that includes things like what is the version of your Extension, what is the title, what privileges does it need to run and lots more.</span></span>

### <span data-ttu-id="9aa50-146">Starten des Erweiterungs Servers</span><span class="sxs-lookup"><span data-stu-id="9aa50-146">Launching the Extension server</span></span>  

<span data-ttu-id="9aa50-147">Wenn Sie die Bereitstellung auf einem Webserver vornehmen, enthält dieser Webserver, ob Apache, IIS, NGINX oder andere, Ihr Webpaket.</span><span class="sxs-lookup"><span data-stu-id="9aa50-147">When you deploy to a web server, that web server, whether it is Apache, IIS, NGINX or any other, contains your web bundle.</span></span>  <span data-ttu-id="9aa50-148">Wenn ein Browser zu einer URL auf einem Server navigiert, `index.html` wird die Datei auf dem Webserver heruntergeladen.</span><span class="sxs-lookup"><span data-stu-id="9aa50-148">When a browser navigates to a URL on a server, the `index.html` file on the web server is downloaded.</span></span>  <span data-ttu-id="9aa50-149">Der Browser navigiert mithilfe von Zertifikaten, Konfigurationsdateien und mehr.</span><span class="sxs-lookup"><span data-stu-id="9aa50-149">The browser navigated using certificates, configuration files, and more.</span></span>  <span data-ttu-id="9aa50-150">Die `index.html` Datei wird an einem bestimmten Speicherort auf dem Webserver gespeichert.</span><span class="sxs-lookup"><span data-stu-id="9aa50-150">The `index.html` file is stored at some special location on the web server.</span></span>   <span data-ttu-id="9aa50-151">Wie funktioniert Ihre Erweiterung?</span><span class="sxs-lookup"><span data-stu-id="9aa50-151">How does your Extension do the same thing?</span></span>  <span data-ttu-id="9aa50-152">Wie wird die Registerkarte Ihres Browsers in der Lage sein, diese ZIP-Datei (Ihre Erweiterung \) zu erhalten?</span><span class="sxs-lookup"><span data-stu-id="9aa50-152">Particularly, how is the tab page of your browser able to get to this zip file \(your Extension\)?</span></span>  <span data-ttu-id="9aa50-153">So funktioniert die Erweiterungs Laufzeit für Sie.</span><span class="sxs-lookup"><span data-stu-id="9aa50-153">That is what the Extension runtime does for you.</span></span>  

<span data-ttu-id="9aa50-154">Die Erweiterung dient für alle Dateien aus der URL \ (Uniform Resource Locator \) unter dem Namen `extension://{some-long-unique-identifier}/index.html` .</span><span class="sxs-lookup"><span data-stu-id="9aa50-154">The extension serves the files all from the URL \(uniform resource locator\) at the name `extension://{some-long-unique-identifier}/index.html`.</span></span>  <span data-ttu-id="9aa50-155">Der Name, den ich in eckige Klammern gesetzt habe, `{some-long-unique-identifier}` ist eine eindeutige Kennung, die der von Ihnen installierten Erweiterung zugewiesen ist.</span><span class="sxs-lookup"><span data-stu-id="9aa50-155">The name I put in brackets, `{some-long-unique-identifier}` is a unique identifier assigned to the Extension that you installed.</span></span>  <span data-ttu-id="9aa50-156">Das bedeutet: Wenn in Ihrem Browser 10 eindeutige Erweiterungen installiert sind, weist jede Erweiterung einen eindeutigen Bezeichner auf, der auf die ZIP-Datei \ (oder das Erweiterungspaket \) verweist, die innerhalb Ihres Browsers installiert ist.</span><span class="sxs-lookup"><span data-stu-id="9aa50-156">That means, if you have 10 unique Extensions installed on your browser, each Extension has a unique identifier that points at the zip file \(or Extension bundle\) installed inside your browser.</span></span>
