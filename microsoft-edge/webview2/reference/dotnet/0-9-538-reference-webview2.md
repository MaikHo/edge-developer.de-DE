# Referenz (WebView2 Core)

Weitere Informationen finden Sie unter [Übersicht über Microsoft Edge WebView2](../../index.md)) und [Erste Schritte mit WebView2](../../gettingstarted/win32.md).  [Microsoft. Web. WebView2. Core.
